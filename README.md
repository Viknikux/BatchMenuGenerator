# BatchMenuGenerator
<img width="859" height="751" alt="image" src="https://github.com/user-attachments/assets/fc820163-dff1-4f5b-8ded-4e0cf519618d" />
A native win32 application that generates batch menus for you!
Compatibility: Works with Windows 11/10 with no extra dependencies, also works on Windows 7 with minor bugs, has no reason to work on linux as batch files arent a thing there, but probably will work with wine/bottles as it is a very simple cpp win32 application.
